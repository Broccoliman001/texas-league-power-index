How Power Score Is Calculated

The Texas League Power Index is a weighted analytical rating designed to measure overall team strength beyond simple win-loss record.

Power Score incorporates:

  • 40% Run Differential Per Game
  • 25% Expected Winning Percentage (X-W/L)
  • 15% Actual Winning Percentage
  • 10% Runs Scored Per Game
  • 10% Runs Allowed Per Game

Teams are normalized against the rest of the league in each category before the weighted score is calculated on a 0–100 scale.

The model intentionally values sustainable performance indicators such as run differential and expected record more heavily than raw standings alone.
